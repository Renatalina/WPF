Uri - класс

Обеспечивает объектное представление универсального кода ресурсов (URI), а также простой доступ к его частям.

Пространство имен:  System

Конструкторы

 Uri(String)	Инициализирует новый экземпляр класса Uri, используя указанный URI.
 Uri(SerializationInfo, StreamingContext)	Инициализирует новый экземпляр класса Uri на основе указанных экземпляров классов SerializationInfo и StreamingContext.
 Uri(String, Boolean)	Устаревшее. Инициализирует новый экземпляр класса Uri с указанным URI и явным управлением преобразованием в escape-последовательность.
 иложения .NET для Магазина Windows	Uri(String, UriKind)	Инициализирует новый экземпляр класса Uri, используя указанный URI. Этот конструктор позволяет указать, является ли строка URI относительным URI, абсолютным URI или неопределенным вариантом.
 иложения .NET для Магазина Windows	Uri(Uri, String)	Инициализирует новый экземпляр класса Uri на основе заданного базового URI и строки относительного URI.
 иложения .NET для Магазина Windows	Uri(Uri, Uri)	Инициализирует новый экземпляр класса Uri на основе сочетания заданного базового экземпляра Uri и относительного экземпляра Uri.
 Uri(Uri, String, Boolean)	Устаревшее. Инициализирует новый экземпляр класса Uri на основе заданных базового и относительного URI с явным управлением преобразованием в escape-последовательность.

Свойства

 AbsolutePath	Возвращает абсолютный путь URI.
 AbsoluteUri	Возвращает абсолютный URI.
 Authority	Возвращает для сервера службу доменных имен (DNS) или IP-адрес и номер порта.
 DnsSafeHost	Возвращает не преобразованное в escape-последовательность имя узла, которое безопасно использовать для разрешения DNS.
 Fragment	Возвращает фрагмент URI.
 Host	Возвращает компонент узла этого экземпляра.
 HostNameType	Возвращает тип имени узла, заданного в URI.
 IsAbsoluteUri	Получает значение, позволяющее определить, является ли экземпляр Uri абсолютным.
 IsDefaultPort	Возвращает значение, позволяющее определить, является ли номер порта, указанный в URI, портом данной схемы по умолчанию.
 IsFile	Возвращает значение, позволяющее определить, представляет ли заданный Uri код URI файла.
 IsLoopback	Возвращает значение, позволяющее определить, ссылается ли заданный Uri на локальный узел.
 IsUnc	Получает значение, позволяющее определить, является ли заданный Uri путем UNC.
 LocalPath	Возвращает представление имени файла в локальной операционной системе.
 OriginalString	Возвращает исходную строку URI, переданную в конструктор Uri.
 PathAndQuery	Возвращает свойства AbsolutePath и Query, разделенные вопросительным знаком (?).
 Port	Возвращает номер порта данного URI.
 Query	Возвращает любые сведения запроса, включенные в заданный URI.
 Scheme	Возвращает имя схемы для данного URI.
 Segments	Возвращает массив, содержащий сегменты пути, которые образуют указанный URI.
 UserEscaped	Указывает на то, что строка URI не была полностью преобразована в escape-последовательность перед созданием экземпляр класса Uri.
 UserInfo	Возвращает имя пользователя, пароль или другие связанные с пользователем сведения, которые соответствуют заданному URI.

Методы

 Canonicalize	Инфраструктура. Устаревшее. Преобразует сохраненный во внутреннем хранилище URI в каноническую форму.
 тся в версии: Приложения .NET для Магазина Windows	CheckHostName	Определяет, является ли указанное имя узла допустимым именем DNS.
 тся в версии: Приложения .NET для Магазина Windows	CheckSchemeName	Определяет, является ли допустимым указанное имя схемы.
 CheckSecurity	Инфраструктура. Устаревшее. Вызов этого метода ни на что не влияет.
 Compare	Сравнивает указанные части двух URI, используя заданные правила сравнения.
 Equals	Сравнивает два экземпляра Uri на предмет их равенства. (Переопределяет Object.Equals(Object).)
 Escape	Инфраструктура. Устаревшее. Преобразует любой небезопасный или зарезервированный знак в компонент пути, используя его шестнадцатеричное представление.
 EscapeDataString	Преобразует строку в ее escape-представление.
 EscapeString	Устаревшее. Преобразует строку в ее escape-представление.
 EscapeUriString	Преобразует строку URI в ее escape-представление.
 Finalize	Позволяет объекту попытаться освободить ресурсы и выполнить другие операции очистки, перед тем как объект будет утилизирован в процессе сборки мусора. (Унаследовано от Object.)
 FromHex	Возвращает десятичное значение шестнадцатеричной цифры.
 GetComponents	Возвращает заданные компоненты текущего экземпляра, используя указанное для специальных знаков указанное escape-преобразование.
 GetHashCode	Возвращает хэш-код для URI. (Переопределяет Object.GetHashCode().)
 GetLeftPart	Возвращает заданную часть экземпляра Uri.
 GetObjectData	Возвращает данные, необходимые для сериализации текущего экземпляра.
 GetType	Возвращает объект Type для текущего экземпляра. (Унаследовано от Object.)
 HexEscape	Преобразует заданный символ в эквивалентное ему шестнадцатеричное число.
 HexUnescape	Преобразует шестнадцатеричное представление символа в сам символ.
 IsBadFileSystemCharacter	Инфраструктура. Устаревшее. Возвращает значение, позволяющее определить, содержит ли имя файловой системы недопустимый символ.
 IsBaseOf	Определяет, является ли текущий экземпляр Uri основой указанного экземпляра Uri.
 IsExcludedCharacter	Инфраструктура. Устаревшее. Возвращает значение, позволяющее определить, нужно ли преобразовывать указанный символ в escape-последовательность.
 IsHexDigit	Определяет, является ли указанный символ допустимой шестнадцатеричной цифрой.
 IsHexEncoding	Определяет, является ли кодировка символа шестнадцатеричной.
 IsReservedCharacter	Инфраструктура. Устаревшее. Возвращает значение, позволяющее определить, является ли заданный символ зарезервированным знаком.
 IsWellFormedOriginalString	Указывает, является ли формат строки, использовавшейся для создания данного Uri, правильным и не требующим дальнейшего преобразования в escape-последовательность.
 IsWellFormedUriString	Указывает, является ли правильным формат данной строки, пытаясь создать на ее основе URI и проверяя, не требуется ли для нее дополнительное преобразование в escape-последовательность.
 MakeRelative	Устаревшее. Определяет разницу между двумя экземплярами класса Uri.
 MakeRelativeUri	Определяет разницу между двумя экземплярами класса Uri.
 MemberwiseClone	Создает неполную копию текущего объекта Object. (Унаследовано от Object.)
 Parse	Инфраструктура. Устаревшее. Анализирует URI текущего экземпляра и проверяет, действительно ли этот код содержит все обязательные части допустимого URI.
 ToString	Получает каноническое строковое представление заданного экземпляра Uri. (Переопределяет Object.ToString().)
 TryCreate(String, UriKind, Uri)	Создает новый Uri, используя заданный экземпляр String и UriKind.
 TryCreate(Uri, String, Uri)	Создает новый Uri, используя заданные экземпляры базового и относительного String.
 TryCreate(Uri, Uri, Uri)	Создает новый Uri, используя заданные экземпляры базового и относительного Uri.
 Unescape	Инфраструктура. Устаревшее. Преобразует указанную строку, заменив все escape-последовательности их представлениями, к которым не было применено escape-преобразование.
 UnescapeDataString	Отменяет преобразование строки в escape-представление.
